Above you will see your work flow menu. In order for your event to go live you will have to complete this process. The menu items will turn from red to green as you complete a step, helping you to track your progress.

Event

This is the area where we can create and edit the main contents of the event, like a hero banner image or the overall goal of the event, or the date and time of the event.

Teams

This is the area where we can create and edit the teams. Events must have teams that participants can be added to. These teams can have their own custom logo and message as well as goals.

Event Pages

This is the area where we can create and edit the event pages. Event pages allows us to create additional pages of information about the event like contacts or where do the donation funds go.

Volunteers

This is the area where we can create and edit the volunteer shifts. The creation of the volunteer page allows the creation of content describing the event as well as adding time slots (start/end) and how many volunteers are needed. This information will be used to create a page that allows be to enroll as a volunteer for a shift.

Sponsors

This is the area where we can create and edit the sponsors. This page allows sponsors to be assigned along with tracking their commitments. These sponsors will be displayed in specific areas on the event pages.

Awards

This is the area where we can create and edit the awards. This page allows us to create content around the awards that will be given for reaching specific goals. Then the individual prizes can be added along with images and descriptions.
